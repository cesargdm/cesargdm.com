# Mexteca

Este proyecto consiste en un web scrapper que buscará las funciones de cine actuales en mi biblioteca de películas local, las guardará en una base de datos y las mostrará de una manera amigable para que la gente pueda encontrar fácilmente lo que quiere ver.

Disponible en [Web](https://cesargdm.github.io/mexteca/) y en la [Play Store](https://play.google.com/store/apps/details?id=com.cesargdm.mexteca).
