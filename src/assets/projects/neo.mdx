# Neo Educate

This app consists of a React Native application being served on all 3 platforms, iOS, Android and Web. It uses Sanity CMS with Mux to serve optimized images and video for it's content.

Taking as inspiration Instagram Stories, Neo delivers bite-sized content to it's users.
